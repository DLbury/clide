Quick Start: Clide + Claude Code MCP

  1. Download — GitHub Releases (Windows: *-setup.exe or *-portable.zip)
  2. SSH profiles — Sidebar → add host, port, user, key/password (credentials stay in app)
  3. Connect — Double-click profile; login in left terminal (password / 2FA / sudo)
  4. Claude Code — Install Claude Code CLI; confirm IDE bridge in Clide sidebar
  5. Ask AI — e.g. "check disk usage"; Claude runs runShellCommand in your shell

Install commands

# Windows portable (no installer)
# Unzip Clide_*_x64-portable.zip → run clide.exe

# Linux .deb
sudo dpkg -i Clide_*_amd64.deb

# Linux AppImage
chmod +x Clide_*.AppImage && ./Clide_*.AppImage

# Register MCP manually (optional)
claude mcp add -s project  # see .mcp.json in repo
