Everything you need

SSH Terminal

Multi-tab shells with split-pane layout. Real-time PTY for local and remote sessions.

Remote Files

SFTP browsing, upload/download, drag-and-drop. Open files directly in Monaco editor.

Claude Code AI

Streaming chat with tool call visualization. Terminal context auto-injected into AI.

MCP Integration

Non-invasive IDE bridge. No shell config pollution. Standard Model Context Protocol.

Resource Monitor

CPU, memory, GPU, disk stats. Independent exec channel, no PTY interference.

Cross-Platform

Windows, macOS, Linux. Built on Tauri 2 for native performance and small footprint.
