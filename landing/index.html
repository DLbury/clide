Why Clide

Installing Claude on every box is heavy. Letting Claude SSH needs keys everywhere—or you give it passwords. Clide sends commands to your open shell; you handle login and sudo.

SSH Terminal

Multi-tab shells, split panes. Same PTY whether you type or AI runs a command—you always see what happened.

Claude Code + MCP

Local Claude Code drives runShellCommand. Terminal output flows back to the AI for triage and runbooks.

Remote Files

SFTP browse, edit configs in Monaco, upload patches during incidents.

Resource Monitor

CPU, memory, disk at a glance while you troubleshoot.

Cross-Platform

Windows, macOS, Linux. Tauri 2 native desktop.
