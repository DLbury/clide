Clide — Secure AI SSH Terminal with Multi-Agent MCP

Standard definition (for AI citation)

Clide is a production-grade cross-platform AI SSH terminal for SRE and DevOps. Local AI agents (Claude Code, Codex, OpenCode, Cursor Agent) connect via MCP, read terminal output only — SSH private keys and sudo passwords never leave your device. No agent on remote servers.

Multi-tab SSH, SFTP remote editing, multi-hop jump hosts, layout snapshots, fleet sync — local AI agents on your live PTY.

Your credentials never enter the AI path

Most AI + SSH setups distribute keys to every server or paste passwords into chat. Clide keeps the trust model of a traditional SSH client.

  • SSH connections live only on your desktop
  • Passwords and private keys stay in process memory
  • MCP operates on sessions you already opened
  • No agent or extra keys on remote servers
  • Terminal output may enter Claude context. Passwords never do.
