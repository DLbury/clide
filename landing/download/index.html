Download Clide — Secure AI SSH Terminal

Free MIT-licensed desktop client for Windows, macOS, and Linux. Includes Windows portable zip (no installer).

GitHub Releases Quick Start FAQ

Windows

  • Installer: Clide_*_x64-setup.exe — in-app updates supported
  • Portable: Clide_*_x64-portable.zip — extract and run clide.exe (v0.1.76+)
  • Requires WebView2 (preinstalled on most Win10/11)
# Portable: unzip, then double-click clide.exe

macOS

  • Clide_*_aarch64.dmg — Apple Silicon
  • Clide_*_x86_64.dmg — Intel

Linux

  • Clide_*_amd64.deb — Debian/Ubuntu
  • Clide_*_amd64.AppImage — universal
sudo dpkg -i Clide_*_amd64.deb
# or
chmod +x Clide_*.AppImage && ./Clide_*.AppImage
