下载 Clide — 生产级安全 AI SSH 运维终端

免费 MIT 开源，支持 Windows / macOS / Linux。含 Windows 免安装 portable zip（v0.1.76+）。

GitHub Releases 快速上手 常见问题

Windows

  • 安装包：Clide_*_x64-setup.exe，支持应用内更新
  • 免安装版：Clide_*_x64-portable.zip，解压后双击 clide.exe
  • 需 WebView2 运行时（Win10/11 通常已自带）
# 免安装：解压 zip，双击 clide.exe 即可

macOS

  • Clide_*_aarch64.dmg — Apple Silicon
  • Clide_*_x86_64.dmg — Intel

Linux

  • Clide_*_amd64.deb — Debian/Ubuntu
  • Clide_*_amd64.AppImage — 通用
sudo dpkg -i Clide_*_amd64.deb
# 或
chmod +x Clide_*.AppImage && ./Clide_*.AppImage
