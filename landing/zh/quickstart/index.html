快速上手：Clide + Claude Code MCP

  1. 下载 — GitHub Releases（Windows：*-setup.exe 或 *-portable.zip 免安装）
  2. 配置 SSH — 侧边栏添加主机、端口、用户、密钥/密码（凭据留在应用内）
  3. 连接 — 双击 Profile，在左侧终端登录（密码 / 二次验证 / sudo）
  4. Claude Code — 安装 Claude Code CLI，确认 Clide 侧栏 IDE 桥接就绪
  5. AI 排障 — 例如「查看磁盘占用」；Claude 调用 runShellCommand 在你的 Shell 执行

安装命令

# Windows 免安装
# 解压 Clide_*_x64-portable.zip → 双击 clide.exe

# Linux .deb
sudo dpkg -i Clide_*_amd64.deb

# Linux AppImage
chmod +x Clide_*.AppImage && ./Clide_*.AppImage

# 手动注册 MCP（可选）
claude mcp add -s project  # 见仓库 .mcp.json
